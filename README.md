# Finanzas Personales

Demo Flutter offline para control financiero personal.

## Ejecutar

Con Flutter instalado y disponible en el `PATH`:

```bash
flutter pub get
flutter analyze
flutter test
flutter run
```

## Alcance actual

- Navegacion principal: Inicio, Calendario, boton central de acciones, Cuentas y Reportes.
- Bottom sheet del boton `+` con acciones financieras iniciales.
- Pantallas demo con datos estaticos.
- Arquitectura base preparada para futuras capas de modelos, base de datos local, servicios y providers.
